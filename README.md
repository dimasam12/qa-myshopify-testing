# 🧪 QA Testing - MyShopify Demo Website

## 📌 Project Overview
This project contains testing documentation for an e-commerce website built using Shopify (demo version). The purpose of this project is to practice and demonstrate Quality Assurance (QA) skills in testing web applications.

---

## 🎯 Objective
- Ensure website functionality works as expected  
- Identify bugs and system errors  
- Practice manual testing and test documentation  
- Build a QA portfolio for entry-level opportunities  

---

## 🔍 Scope of Testing
The following features were tested:

- ✅ Login  
- ✅ Register  
- ✅ Product Listing  
- ✅ Add to Cart  
- ✅ Cart Page  
- ✅ Checkout (limited - demo)  
- ✅ Form Validation (positive & negative cases)  

---

## 🧪 Testing Type
- Manual Testing  
- Functional Testing  
- Positive Testing  
- Negative Testing  

---

## ⚙️ Tools Used

- Git & GitHub
